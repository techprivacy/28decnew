XSender - Mass Email and Messaging Marketing Tools

Overview

XSender is a mass email and text message sending web applicaiton tools for sending text message via 4 popular sms API gateway and using most of powerful android sms sender app which supports from android version 5.0 and also your own country mobile netowrk sim for sending sms. Start you own SaaS platform without any restrictions for sending sms through android device.

You can manage your customer database from XSender mordern admin panel and also buy dynamic subcription plan for SMS and Email plan via different payment gateway.

XSender manage bulk requests smoothly using a powerful Laravel Queue system and also manage server performance.

Server Requirements

  • ⊛ PHP Version 7.4 or above
  • ⊛ MySQL Version 5.7+ or MariaDB version 10.2+
  • ⊛ Ctype PHP Extension
  • ⊛ Fileinfo PHP extension
  • ⊛ JSON PHP Extension
  • ⊛ Mbstring PHP Extension
  • ⊛ OpenSSL PHP Extension
  • ⊛ PDO PHP Extension
  • ⊛ PDO_MYSQL PHP Extension
  • ⊛ Tokenizer PHP Extension
  • ⊛ XML PHP Extension
  • ⊛ CURL PHP Extension
  • ⊛ GD PHP Extension

How To Install on Local and Hosting Server

XSender web application installation process is very easy just follow the instructions:

  • Step-1: Create a database into MySQL server using phpMyAdmin or MySQL Command line
  • Step-2: Unzip the Xsender project.zip file into your web server root or htdocs or subdomain directory
  • Step-3: Now go to browser then put url for installing script like this:-http://your-domain.com/install or if localhost then http://localhost/{project_directory}/install
  • Step-4: In this step software check server requirements and permissions for final installation and give all the info
  • Step-5: After successfully install set up all basic settings from admin panel

    CRON Job Setup

    You have installed the software successfully. Now set a cron jobs on your server for an autometic support from Android SMS Gateway App, Plan Subscription, SMS Schedule, Email Schedule, and Queue activity.

    Set all the cron jobs to your hosting following the command links. You can also copy/paste the links from admin panel > System Settings > Settings > Modal
    curl -s http://your-domain-name/android/api/cron
    curl -s http://your-domain-name/subscription/cron
    curl -s http://your-domain-name/schedule/email/cron
    curl -s http://your-domain-name/queue-work
